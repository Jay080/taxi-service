## Tech Stack

- **Backend:** Go (Control Plane), Rust (Data Plane)
- **ML Pipeline:** Python + Rust
- **Real-time Processing:** Rust + CUDA
- **Storage:** TimescaleDB + CockroachDB
- **Mobile:** Flutter
- **Infrastructure:** Kubernetes
- **External Services:** 2GIS, SMS Authorization
